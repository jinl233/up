## 转置矩阵

  1. $(A^T)^T = A$
  2. $(A + B)^T = A^T + B^T$
  3. $(kA)^T = kA^T$
  4. $(AB)^T = B^T A^T$
  5. $|A^T|=|A|^T=|A|$

## 对称矩阵

  1. 对称矩阵的特征值都是实数。
  2. 对称矩阵的特征向量可以正交化，即对于实对称矩阵 $A$，存在正交矩阵 $P$，使得 $P^T A P$ 是对角矩阵。
  3. $A$ 和 $B$ 都是对称矩阵，则 $A+B$ 和 $A-B$ 也是对称矩阵。
  4. $A$ 是对称矩阵，$c$ 是标量，则 $cA$ 也是对称矩阵。

## 逆矩阵

  1. $(A^{-1})^{-1} = A$
  2. $(AB)^{-1} = B^{-1} A^{-1}$
  3. $(A^T)^{-1} = (A^{-1})^T$
  4. $(kA)^{-1}=k^{-1} A^{-1}$
  5. $|A^{-1}|=|A|^{-1}= \frac{1}{|A|}$

## 伴随矩阵

$$A^* =|A|A^{-1}$$
  1. $(kA)^{*}=k^{n-1} A^{*}$
  2. $A^T(A^T)^*=|A|E$
  3. $(A^{-1})(A^{-1})^* = |A|E$
  4. $(AB)^* =B^* A^*$
  5. $A^{*}(A^{*})^* =|A^{*}|E$
  6. $(A^{*})^*=|A|^{n-2}A$
